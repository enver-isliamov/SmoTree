## Во функции которые не работают:
****************
1. Смена версии (v2 -> v1)	✅ Реализовано	Загрузите 2 версии в ассет. Переключитесь через бейдж. Плеер должен моргнуть (перезагрузиться) и показать правильное видео. Внизу появится уведомление Loaded Version X. [не преключается между версиями]
2. Индикация файла	✅ Реализовано	В шапке плеера рядом с названием ассета должно отображаться имя файла (например, video_v1.mp4) и плашка DRIVE или LOCAL. [название в плеере Файлик.mp4 а название на диске Файлик_v1.mp4]
3. Переименование папки	✅ Реализовано	Зайдите в Дашборд. Нажмите "Edit" на проекте. Измените имя. Зайдите на Google Диск — папка проекта должна сменить имя автоматически [на диске название не меняются]
4. Проверка удаленных файлов	✅ Реализовано	Удалите файл версии вручную с Google Диска (в корзину). Откройте эту версию в плеере SmoTree. Вместо вечной загрузки должен появиться красный экран: "File Deleted from Drive". [нет рабоетет  и отображет Loaded Version 4: Reels-Hudenie.mp4]
5. Очистка "битых" версий	✅ Реализовано	На красном экране ошибки нажмите кнопку "Remove Version from App". Версия должна исчезнуть из списка, и вас перекинет назад [нет возможности протестировать потому что предыдущие пункты не работают]
